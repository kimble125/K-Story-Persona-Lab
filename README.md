# K-Story Persona Lab
### 조선시대 한글편지로 본 '관계에 따라 달라지는 말투' — 공공 문화데이터 기반 말투 정량 분석

> **제4회 문화체육관광 인공지능·데이터 활용 공모전 · 데이터 분석 분야 출품작**
> 국립국어원 국어 역사 자료 말뭉치(언간)를 분석해, 한국인의 말투(경어)가 **상대와의 관계에 따라 어떻게 달라지는지**를 데이터로 규명한다.

## 1. 문제 인식
 
AI·콘텐츠 시대에 '한국적 캐릭터·페르소나'는 K-콘텐츠 경쟁력의 핵심 자산이 되고 있다.
그러나 정작 우리 역사·문학 인물의 **'말투'는 한 번도 데이터로 정리·유형화된 적이 없다.**
사극 대사, 캐릭터 어조, 교과서 속 인물의 말씨는 대체로 작가의 *감(感)*과 관습에 의존한다.
 
특히 한국어의 정체성인 **경어법**은 "무엇을 말하는가"보다 "**누구에게 말하는가**"에 의해 좌우된다.
그런데 이 *관계–말투*의 상관은 지금까지 정성적 직관으로만 이야기돼 왔다.
게다가 옛 문헌의 언어(고어)는 현대 형태소 분석기로 처리되지 않아 **데이터 분석의 사각지대**였다.

> **"역사 인물의 말투를 데이터로 바꿀 수 있다면?"** — 이라는 의문이 이 프로젝트의 출발점이다.

## 2. 핵심 질문과 가설
 
- **질문**: 조선시대 사람들은 상대(수신자)와의 관계에 따라 말투(경어)를 실제로 바꾸었는가? 바꿨다면 얼마나, 어떻게?
- **가설**: 말투는 화자의 고정된 속성이 아니라, **수신자와의 관계에 따라 달라지는 '관계의 함수'**다.
- 
## 3. 핵심 발견 — 가설 입증
 
국립국어원 언간(한글편지) **223통**을, 발신자가 수신자에게 갖는 **관계 방향**(상향·부부·하향)으로
나누어 경어 사용을 측정했다.
 
| 관계 방향 | 편지 수 | 高경어 종결 비율 |
|---|---:|---:|
| **상향** (손아래→손위) | 60 | **0.144** |
| 부부 | 48 | 0.091 |
| **하향** (손위→손아래) | 113 | **0.022** |
 
- 손위에게 쓸 때가 손아래에게 쓸 때보다 **약 6.5배** 더 정중했다.
  차이는 통계적으로 매우 유의하다 — **Kruskal–Wallis H = 61.1, p = 5.5 × 10⁻¹⁴**.
- **주체높임(–시–)은 관계와 무관하게 평평**했다. 즉 *누구를 높이느냐(주체)* 와
  *누구에게 말하느냐(청자)* 는 **분리된 축**으로 작동한다. 단순한 '높임말 많다/적다'가 아니다.
- **같은 사람도 상대에 따라 말투를 바꿨다(within-person).**
  예) 안동권씨 — 며느리로서 高경어 **0.40** vs 장모로서 **0.025**.
![관계별 경어](results/figures/deference_by_relationship.png)
 
> 결론: **말투는 관계의 함수다.** 이는 단순 통계를 넘어, *페르소나를 관계의 함수로 모델링해야 한다*는
> 콘텐츠 설계 원리로 이어진다.

## 4. 접근 방법 (왜 이렇게 분석했나)
 
1. **고어 호환 글자·종결어미 단위 스타일로메트리.**
   konlpy·Mecab 등 현대 형태소 분석기는 고어에서 깨진다. 그래서 형태소 분석기를 쓰지 않고,
   글자·어휘·종결부 패턴으로 **17개 피처**(高경어 종결 비율, 주체높임 –시– 비율, 문장 길이,
   어휘 다양성, 한자 의존도, 종결어미 분포 등)를 산출한다. **모든 피처는 설명 가능**(심사 요건).
2. **관계 주석 활용 — 이 데이터만의 강점.**
   언간은 발신자 역할(`sender="임영(아들)"`)이 주석돼 있다. 이를 상향/부부/하향으로 분류해
   *관계 → 말투* 의 비교를 가능케 한다.
3. **통계·군집.** 관계 방향별 평균 비교 + Kruskal–Wallis 검정, KMeans 군집(실루엣 기반 k 선택),
   PCA 2D 시각화.
4. **챗봇 없음.** 이 저장소는 순수 *데이터 분석*이다. 대화 생성·AI 캐릭터 평가는 범위 밖.
   (데이터분석 부문 평가 기준 — 기획성·논리성·구체성·창의성·활용성 — 에 집중.)
   
## 5. 파이프라인 & 저장소 구조
 
```
data/raw/*.xml (언간)
  └─ src/data/load_historical.py         TEI/XML 파싱 (발신자·수신자·연도)
  └─ src/analysis/relationship_style.py  letter 단위 + 관계 방향 분류 + 경어 피처   ← 핵심
        └─ scripts/run_relationship_analysis.py  관계별 경어·within-person·군집·차트·리포트
  └─ src/features/stylometry.py          17개 문체 피처 (보조: 문서 단위 분석)
  └─ src/cluster/persona_typology.py     표준화 → KMeans(실루엣) → 유형
  └─ src/viz/                            PCA·관계 차트
results/ (CSV·figures) + reports/ (분석 리포트)
```
 
| 경로 | 내용 |
|---|---|
| `src/analysis/` | **관계 기반 말투 분석(핵심 모듈)** |
| `src/features/stylometry.py` | 고어 호환 17 피처 |
| `src/cluster/` | KMeans 페르소나 유형화 |
| `scripts/` | 분석 드라이버 · 말뭉치 다운로드 |
| `docs/` | SPEC · DATA_SOURCES · ADR · 핸드오프 문서 |
| `data/sample/` | 공개 도메인 샘플(말뭉치 없이 동작 검증) |

## 8. 활용과 확장 (활용성)
 
- **콘텐츠**: 사극·웹툰·게임의 역사 인물 대사 고증. 캐릭터의 '말투'를 *상대별로 차등 설계*하는 정량 근거.
- **교육**: 데이터로 보여주는 한국어 경어법 — "왜, 누구에게 경어를 쓰는가"를 실제 사례로.
- **아카이브**: 역사 인물 말투의 정량 자산화.
- **확장 로드맵**:
  1. 역사 2023(고소설)·현대 일상대화 **대조** → 시대·매체별 경어 변화의 일반화
  2. 실록·인명정보 **연계** → 발신자 신분·관직·성별별 말투 차이
  3. 관계→경어 매핑을 **캐릭터 말투 설계 가이드**로 제품화(포트폴리오)
