# ADR-0001: Data-analysis stylometry (A안), no chatbot

- **Status:** Accepted (locked)
- **Context:** Entry for 데이터분석 track. Original idea was character-AI evaluation (chatbot-adjacent).
- **Decision:** Pure data analysis — quantify/typologize 말투·페르소나 via stylometry → clustering.
  No chatbot, no LLM dialogue generation in the core.
- **Evidence:** In 2023–2024 winning case collections, "추천" 31–38×, "챗봇"≈1 (never a data-analysis
  winner), "페르소나"=0. A chatbot scores ~0 in this track.
- **Consequences:** Chatbot/demo deferred to an optional 우수사례 upgrade (Stage 2). Repo README
  rewritten from the old chatbot-eval framing to A안.
